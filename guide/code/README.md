# Bytecode Technique Code Samples

This folder contains minimal before/after Java examples for each bytecode-size technique.

Each technique subfolder has:
- `Before.java`
- `After.java`
- `javap_diff_default.diff` (generated by the runner)

## Run All Techniques

```bash
python3 guide/code/run_techniques.py --save-javap-diffs
```

This compiles all technique folders and prints a table with:
- total `.class` bytes (`Class Before`/`Class After`)
- raw-deflate compressed bytes (`Deflate Before`/`Deflate After`)

The deflate metric is only a directional hint. Real JAR compression can vary with project shape, resource mix, and packaging order.

## Run One Technique In Isolation

```bash
python3 guide/code/run_techniques.py --technique 10_instanceof_pattern_if --save-javap-diffs
```

## Compare Across JDKs

Pass one or more compiler commands with labels:

```bash
python3 guide/code/run_techniques.py \
  --javac jdk17=/path/to/jdk-17/bin/javac \
  --javac jdk21=/path/to/jdk-21/bin/javac \
  --javac jdk26=javac
```

Optionally force a language target:

```bash
python3 guide/code/run_techniques.py --release 21
```

If a technique uses language features unsupported by a selected JDK, that row is reported as failed with the compiler error.
